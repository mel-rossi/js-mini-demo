The easiest part of the demo was using variables. Variables are great, extremely convinient to have for all sorts of things in code. Makes calculations easy. Though all in all, everything else wasn't particularly difficult either. The hardest part of the demo was probably just getting used to how the objects and methods in JS work. It's pretty similar to other object-oriented languages, so it's just getting used to the specific habits of the language in comparison to others. The Date Object provides a value of time. Different Date Methods can be used to do certain things with, extract specific parts of and format said time. The Number Object represents floating-point numbers. Same as a the Date Object different Number Methods can be used to manipulate said numbers. Displaying results in the browser is as simple as using the Document Object and connecting the elements in JS and HTML through an id.
